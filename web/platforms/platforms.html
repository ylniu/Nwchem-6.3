NWChem Supported Platforms

NWChem is supported on at least the following platforms, and is readily ported to essentially all sequential and parallel computers.

  • IBM SP and Power3/4/5 system running AIX
  • SGI IRIX SMP systems
  • SUN workstations running SOLARIS
  • HP PA-RISC workstations running HPUX
  • Fujitsu VX/VPP
  • Fujitsu PRIMEPOWER
  • HP Alpha SMP servers running Tru64 or Linux.
  • HP Alpha SC series.
  • Workstation networks
  • IA32-based workstations running Linux or FreeBSD
  • IA32 Linux Clusters with Giganet switch using the VIA protocol
  • IA32 Linux Clusters with Myrinet switch using the GM software
  • IA32-based PC running Microsoft 32-bit Windows
  • PowerPC workstations running Linux
  • Itanium servers running Linux or HPUX
  • Linux IA32, Itanium and Alpha clusters with Elan3 switch
  • Itanium Linux Clusters with Myrinet switch using the GM software
  • SGI Altix systems
  • Apple Mac computers using OS X version 10.3 ("Panther") or later
  • Linux Itanium clusters with Elan4 switch
  • Linux AMD Opteron computers
  • Apple Mac Intel computers
